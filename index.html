The Gaming Development summer camp gives you a foundation to start on a path towards a career in computer programming. Programming is hard. Skills Covered

Each program covers different areas of the field of technology

  • Gaming Development

    In this course, students use Microsoft TouchDevelop to create games and learn to think like a programmer. After taking this course, students will be prepared to sit for Microsoft's exam 98-380: Introduction to Programming Using Block-Based Languages. This exam measures the student's ability to apply programming strategies to solve computational problems, design algorithms, work with data structures, and more.

  • Cyber Security

    In this course, students learn the basics of how computers work, how the internet works, how computers communicate with one another all over the world, they install their first server, they gain an understanding how to protect themselves from hackers in a corporate setting, and learn how to set up company networks. This course leads to the Logical Operations CyberSafe certification, and on onr first year we had over 80% passing rate.
